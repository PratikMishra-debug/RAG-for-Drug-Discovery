# Drug RAG: Reducing Hallucination in LLM-Assisted Drug Discovery

## Overview

This project explores Retrieval Augmented Generation (RAG)
to reduce hallucination in Large Language Models for drug-related
question answering.

## Tech Stack

- Python
- LangChain
- ChromaDB
- Sentence Transformers
- Ollama
- Llama 3.2

## Architecture

User Query
↓
Embedding Model
↓
Vector Database
↓
Retriever
↓
LLM
↓
Answer

- Dataset loading completed

- Local LLM integration completed
- RAG pipeline implemented
